# イキイキめーたー リリースノート

---

## v1.0.0 — 2026年3月29日

### はじめに

「イキイキめーたー」は、あなたの理想の1日と現実の1日のズレを「生活効率スコア」で見える化するライフゲームです。通勤できた、ゆっくり休めた、少し勉強した——そんな当たり前の行動をすべてポイント化し、毎日の充実度をメーターで確認できます。

本バージョンは初回公開リリース（v1.0.0）です。

---

### 主な機能

#### スコアシステム
- 1日の行動を「時間pt」「位置pt」「タスクpt」の3軸で評価し、総合スコアをメーターで表示
- タスクモードは3段階（イージー30pt・ハーフ50pt・ハード100pt）から選択可能
- 早起きボーナス：起床時刻に応じて最大30ptを加算（4時台30pt / 5時台25pt / 6時台20pt / 7時台10pt）

#### 今日のモード
- **通常モード**（出勤日）、**休日モード**、**出張モード**、**病欠モード**の4種類に対応
- 設定で休日曜日を登録すると、該当日は自動的に休日モードで起動
- 出張・病欠モードは移動ログなし（休日モードと同じタスク構成）

#### タスク記録
- 各タスクのデフォルト表示は「何をしましたか？」（未選択状態）
- 選択肢に「何もしなかった 💤（0pt）」を追加
- 予定時刻の20分以上先のタスクは🔒でロック（未来のタスクは入力不可）
- 過去・現在のタスクは後から入力可能

#### 週間グラフ
- 過去7日間のスコア推移を棒グラフで表示
- 今日の記録をリアルタイムで反映

#### 生活スタイル設定
- 起床・就寝・出勤・退勤・昼休みなどの時刻を自由に設定
- 休日曜日の登録（複数選択可）
- タスクモードの切り替え

#### 使い方ガイド
- アプリ内に詳細な使い方ページを搭載
- iPhone・Androidのホーム画面への追加手順を掲載

### 公開・配布対応

- **PWA対応** — スマホのホーム画面に追加してアプリとして使用可能
- **OGP設定** — SNSシェア時にタイトル・説明文・アイコンが表示される
- **プライバシーポリシー・利用規約** — アプリ内に専用ページを設置
- **初回同意画面** — 初回アクセス時にプライバシーポリシーへの同意を求めるモーダルを表示
- **クレジット表示** — 全ページ最下部に「© 2026 [蓮華堂](https://rengedo.asia/)」を表示

### 既知の制限事項

- データはブラウザのローカルストレージに保存されます。ブラウザのデータ消去やキャッシュクリアを行うと記録が失われます。
- 夜勤や変動勤務の方には現在未対応です。
- 複数端末間でのデータ同期には対応していません。
